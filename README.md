## ZenGuard
Better Server Protection.

# Features
* OP-Protection. No more backdoors!
* Protected-Commands. Blacklist grief commands, (eg. /ban Owner)
* No Depencies
* Very configurable
* Fast config reload
* Player data saving (Optional)
* Custom GUI
